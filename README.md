# 🏋️ Churn Gym Detector

Sistema web para la detección temprana de abandono (churn) en clientes 
de gimnasios mediante análisis de actividad.

## Autores
| Nombre | Rol |
|---|---|
| Diego Forteza Benito | Backend — Spring Boot, API REST, BD |
| Javier Escudero García | Frontend — Vanilla JS / HTML / CSS |

## Stack tecnológico
- **Backend**: Java 21 + Spring Boot 3 + PostgreSQL
- **Frontend**: Vanilla JS / HTML5 / CSS3
- **Infraestructura**: Docker + Docker Compose

## Cómo ejecutar el proyecto
```bash
docker-compose up --build
```
- Frontend: http://localhost:3000  
- Backend API: http://localhost:8080/api  
- Base de datos: puerto 5432
